Sobre a Aplicação

Conversor de Arquivos

Uma aplicação web moderna e eficiente para conversão de arquivos, similar ao ILovePDF e Sejda.

Características

  • Conversão Rápida: Processamento eficiente de arquivos
  • Múltiplos Formatos: Suporte para PDF, DOCX, TXT, JPG, PNG e mais
  • Interface Intuitiva: Design moderno e fácil de usar
  • Segurança: Processamento seguro dos seus arquivos
  • Responsivo: Funciona em desktop, tablet e mobile

Tecnologias Utilizadas

Backend
  • Java 17
  • Spring Boot 3.2.0
  • Spring MVC
  • Apache PDFBox
  • Apache POI
Frontend
  • Thymeleaf
  • Bootstrap 5
  • JavaScript
  • Bootstrap Icons

Arquitetura

A aplicação segue as melhores práticas de desenvolvimento, utilizando arquitetura em camadas:

  • Controller Layer: Gerenciamento de requisições HTTP
  • Service Layer: Lógica de negócio e conversão de arquivos
  • Model Layer: Entidades e DTOs
  • Exception Handling: Tratamento centralizado de erros
Dica: Esta aplicação foi desenvolvida seguindo os princípios SOLID e Clean Code, garantindo código limpo, manutenível e escalável.
